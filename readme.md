# GIS Notebook
Jupyter Notebook container image for GIS.

## Software Inluded
- beautifulsoup4
- boto3
- eo-learn
- geopandas
- geoplot
- laspy
- lightgbm
- natsort
- netcdf4
- networkx
- psycopg2
- pysal
- rasterio
- rclone
- selenium
- sentinelhub
- xeus-python

This image is based upon the [Jupyter Stack PyTorch image](https://github.com/jupyter/docker-stacks/tree/main/images/pytorch-notebook)
